# CNBS Reference Reconciliation

## Objetivo

Comparar agregados canónicos contra el workbook de referencia sin promoverlo a fuente primaria operacional.

## Cuándo usarla

- al evaluar publicación de una nueva versión

## Entradas esperadas

- agregados canónicos
- valores de referencia y período detectado

## Reglas

- distinguir discrepancia real de discrepancia por período

## Validaciones

- totales
- subtotales
- rankings

## Errores comunes

- reconciliar períodos distintos como si fueran idénticos

## Criterios de salida

- reporte con severidades y decisión de publicación
