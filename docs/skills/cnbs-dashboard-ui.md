# CNBS Dashboard UI

## Objetivo

Mantener una UI institucional sobria, rápida y clara para consulta pública y validación administrativa.

## Cuándo usarla

- al diseñar vistas públicas y privadas

## Entradas esperadas

- agregados publicados
- metadata de versión

## Reglas

- no recalcular lógica pesada en cliente
- separar vista ejecutiva y analítica

## Validaciones

- responsive
- accesible
- consistente con metadata publicada

## Errores comunes

- depender del Excel en runtime

## Criterios de salida

- pantallas listas para consumo institucional
